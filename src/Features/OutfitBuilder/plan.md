# Outfit Visualizer

## Goal

A user can see an image of themselves and then use that image in an app where they can see the clothes in their closet on their avatar with the clothes that is in their closet

Photos from closet, should be able to be super imposed on avatar. And mix and match: a 'top' should got on the top of avatar,a 'bottom' should go on the bottom of the avatar

## Functionality

User can upload photo as their avatar

User can see their closet

User can select from closet categories 

Items overlay on top of avatar in approriate spots tops on top bottoms on bottom, shoes on feet, dresses over entire body 

use image detector API to accurately get body measurements, 

## Layout

Three panels -
top: days weather
right: saved outfits, 
middle avatar,
left: closet,
